For this time here is an attempt of implementing IHierarchicalData support, since spark DataGrid only support IList as dataProvider I have written a little adapter that implement IList and ICollectionView that allows to get data from a IHierarchicalData as  source.
This implementation has been greatly inspired by the Alex Harui one.

sample | source

note : this implementation has not been tested enough and can contain bugs.

This is for the moment the last part of my experiments with spark DataGrid, the next feature I would like to implement is a ItemRenderProvider equivalent of the mx AdvancedDatagrid , however this seems to be a lot of work, and I do not know if I can accomplish it without completely refactoring the spark Grid component (what I absolutely do not want to do).
I'll perhaps try in the future  to improve my implementation and provide a well tested component, or to add new features.
