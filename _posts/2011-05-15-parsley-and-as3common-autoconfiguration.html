Combining AS3Commons bytecode and Parsley can produce very interesting concepts that greatly help in developing Flex application.

Playing a little with this 2 libraries, I found an interesting way of replacing configuration file  when building a Parsley context :

By searching class with an ObjectDefinition Metadata in the application loaderInfo, I automatically detect class that should be included in parsley context, and so avoid unnecessary configuration file.
